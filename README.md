# 🌟 Портфолио Димы — Одностраничный сайт-визитка

> «Громко. По делу.»

Этот проект — стильное, интерактивное портфолио, созданное для творческого человека. Сайт сочетает в себе минималистичный дизайн, приятные анимации и необычные интерактивные элементы, которые делают просмотр запоминающимся.

## 📸 Внешний вид

- **Фоновое изображение** — ночной пейзаж со звёздами (взято с Unsplash).
- **Кастомный курсор** — светящаяся точка, которая плавно следует за мышью.
- **Блёстки** — при движении мышью за курсором появляются искрящиеся частицы, которые быстро исчезают.
- **Падающие звёзды** — по всему экрану разбросаны золотистые звёздочки (★). При клике на любую из них она с анимацией падает вниз, вращаясь, а на её месте появляется новая. Всего на экране **65 звёзд**, они падают **в 2 раза медленнее** обычного (длительность анимации — 1.8 секунды).

## 🧩 Функциональные блоки

- **О себе** — краткое описание (например, «Люблю отдыхать»).
- **Умения** — перечень навыков (в демо-версии указано «что-то умею»).
- **Работы** — список проектов (например, «этот сайт и что-то ещё»).
- **Что я люблю** — раздел с текстом «кушать яблочки», стрелкой вниз и изображением яблока (`apple.jpg`).
- **Связь** — контактный email (whyddreamerer@gmail.com).
- **Моя песня** — встроенный аудиоплеер для локального MP3-файла (`You've No Choice.mp3`). При нажатии на кнопку плеер плавно раскрывается.

## 🛠️ Технологии

- **HTML5** — семантическая разметка.
- **CSS3** — кастомные свойства (CSS-переменные) для поддержки светлой/тёмной темы, анимации, флексбоксы и гриды.
- **JavaScript (ES6)** — интерактивность: кастомный курсор, блёстки, падающие звёзды, управление плеером и переключение тем.
- **Адаптивность** — корректно отображается на всех устройствах (мобильные, планшеты, десктопы).

## 📁 Структура файлов

Проект состоит из одного HTML-файла, в который встроены все стили и скрипты. Для работы потребуются следующие медиа-файлы (расположите их в той же папке):

```
/
├── index.html               # Главный файл
├── apple.jpg                # Изображение яблока (для раздела «Что я люблю»)
├── You've No Choice.mp3     # Аудиофайл песни
└── README.md                # Этот файл
```

## 🚀 Как запустить

1. Скачайте все файлы в одну папку.
2. Убедитесь, что имена файлов совпадают с указанными выше (регистр важен).
3. Откройте `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).
4. Наслаждайтесь просмотром!

## 🎨 Особенности дизайна

- **Тёмная тема по умолчанию** с возможностью переключения на светлую (сохраняется в `localStorage`).
- **Плавные анимации** — все переходы, появление элементов и падение звёзд происходят мягко.
- **Минималистичные карточки** с полупрозрачным фоном и размытием (backdrop-filter) — контент остаётся читаемым поверх фонового изображения.
- **Акцентный цвет** — тёплый золотистый оттенок, который используется в кнопках, тенях и свечениях.

## ✨ Дополнительно

- **Песня** — вы можете заменить `You've No Choice.mp3` на свой трек, сохранив то же имя или изменив путь в теге `<source>`.
- **Картинка яблока** — замените `apple.jpg` на своё изображение, если хотите.
